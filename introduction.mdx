## Key Features

### IT Service Management

- **Ticket System**: Comprehensive ticket management for incidents and service requests
- **SLA/OLA Management**: Define and track service level agreements with automatic escalation
- **Validation Workflows**: Multi-level approval processes for tickets and changes
- **Satisfaction Surveys**: Automated user satisfaction tracking
- **Recurrent Tickets**: Schedule recurring maintenance and tasks

### Asset & Inventory Management

- **Hardware Inventory**: Track all IT assets including computers, monitors, peripherals, and network devices
- **Software Management**: License tracking and software deployment management
- **Financial Management**: Cost tracking, budgets, and contract management
- **Automatic Discovery**: Integration with inventory tools for automatic asset updates

### Collaboration & Communication

- **Email Notifications**: Customizable email alerts for ticket updates and assignments
- **Internal Notes**: Private comments for technician collaboration
- **Document Management**: Attach and organize files related to tickets and assets
- **Observers & Groups**: Flexible actor assignments with user and group support

### Customization & Integration

- **Plugin System**: Extend functionality with community and custom plugins
- **REST API**: Full-featured REST API for external integrations
- **Business Rules**: Automated workflows based on ticket properties
- **Templates**: Predefined ticket templates with mandatory and hidden fields
- **Multi-Entity**: Support for multiple organizations in a single instance

## Modern Technology Stack

ITSM-NG is built with proven, modern technologies:

<Note>
  ITSM-NG requires PHP 8.1+ and MariaDB/MySQL, ensuring compatibility with current hosting environments and security standards.
</Note>

- **Backend**: PHP 8.1+ with Symfony components, Twig templating, and Doctrine
- **Frontend**: Bootstrap 5, jQuery, and modern JavaScript libraries
- **Database**: MariaDB 10.0+ or MySQL 5.6+
- **API**: RESTful API with JSON responses
- **Security**: Sodium encryption for sensitive data, CSRF protection, and secure session management
